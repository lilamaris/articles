## 개요

**산업용 무선 센서 네트워크**(Industrial Wireless Sensor Networks, IWSNs)에서 **데이터 신선도**(freshness)를 보장하는 **AoI**(Age of Information) 기반 스케쥴링 문제를 다룬다.

### AoI(Age of Information)
- **정보가 얼마나 최신인가**를 나타내는 메트릭
- 데이터 생성 시점부터 수신 완료까지 전체 과정을 고려
- 제어 및 모니터링 시스템에서 실제 사용할 수 있는 최신의 데이터 유효성을 측정

기존 연구는 시스템 전반의 장기 평균 AoI를 최소화하는데 집중했음.
그러나 평균 AoI를 최소화하는 노력이 모든 패킷의 최대 AoI를 제한할 수 있다는 것은 아님.
어떤 노드의 특정 데이터가 지정된 임계값을 넘는 순간, 제어 고장이나 성능 저하가 일어날 수 있음

-> 개별 패킷의 AoI가 정해진 범위 안에 들어오도록 보장하는 스케쥴링 접근을 제안함

## AoI 경계를 보장하는 스케쥴링 알고리즘
장기 평균 AoI를 최적화하는 대신 각 노드가 샘플링한 데이터 패킷의 AoI를 미리 정해진 경계 범위 내 있도록 스케쥴러가 의사결정을 내린다.

## 전제

### 시간 슬롯
- IEEE 802.15.4 무선 표준을 전제로, 센서 노드는 고정 길이 슬롯 동안 패킷을 전송할 수 있다

### 단일 패킷 큐 모델
- 에너지 제약과 단순성을 이유로, 각 노드는 큐에 하나의 패킷만 유지한다.
- 새 패킷이 만들어지면, 기존에 큐에 유지하던 패킷은 버려진다.
- 샘플이 전송되기 전에 다음 샘플이 만들어지면 데이터가 유실된다.

---

time-slotted IWSN 에서 하나의 sink 노드와 N개의 소스 노드가 싱글 홉으로 구성되어있다.

하나의 센서 노드 N_i는 T_i 주기마다 데이터를 샘플링하고, I_i마다 샘플링한 데이터를 sink 노드로 전송한다.
